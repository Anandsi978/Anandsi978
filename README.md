## 🎯 About Me

I'm a final-year B.Tech student specializing in **Cloud Computing and Machine Learning** at Babu Banarasi Das University, graduating in 2026. I like taking things from "works on my laptop" to "runs on AWS" — deploying, breaking, debugging, and deploying again.

Most of my hands-on experience lives at the intersection of **backend logic and cloud infrastructure**: writing Python and Java that does something useful, then wiring it up to EC2, S3, and Lambda so it actually stays alive after I close my terminal.

Outside of code, I've coordinated technical events as part of my college's Technical Club Board and led cross-functional teams as General Secretary of the Central College Forum — so I'm equally comfortable in a codebase or a planning meeting.

**Right now I'm looking for:** Software Developer · Cloud Engineer · Associate Software Engineer · Technical Support Engineer roles.
